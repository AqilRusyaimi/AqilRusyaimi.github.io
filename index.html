Interaction & State Management

Exploring state management in Compose, including state hoisting, remember, and user interaction handling.

Reflection
Managing state was a little confusing, but thanks to the lab tutorial I understood how to handle it. I also used AI (Gemini) to help fix errors I encountered during development.

Material Design

Applying Material Design 3 principles and generating cohesive colour palettes using the Material Theme Builder.

Reflection
This lab was really challenging since I'm not strong in design. Fortunately, the Material Theme Builder helped me generate a harmonious, industry-standard colour palette by selecting just a single primary colour.

Navigation

Implementing multi-screen navigation with Navigation Compose, passing arguments between destinations.

Reflection
This lab was very challenging due to multiple interconnected files, but I managed to complete and understand it. I used AI assistance to learn how to pass multiple data arguments between screens rather than a single value.
Lab 5

Room Database

Integrating a Room local database with a DAO, entities, and ViewModels to persist and display data.

Reflection
This final lab was the most challenging, containing multiple files that needed to be linked with the UI screens. I relied on AI to identify and resolve errors encountered throughout the process.
